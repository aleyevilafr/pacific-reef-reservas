# Sistema de Reservas Hoteleras Pacific Reef

Este proyecto corresponde al diseño de un sistema de gestión de reservas hoteleras, desarrollado en el contexto de la asignatura de Ingeniería de Software.

## Descripción

El sistema permite a los usuarios consultar disponibilidad de habitaciones, realizar reservas y efectuar pagos en línea. Además, contempla funcionalidades de administración como la gestión de reservas y generación de reportes.

Este trabajo se enfoca en el diseño del sistema utilizando metodologías ágiles (Scrum) y diagramas UML.

## Contenido del repositorio

### uml/
Contiene los diagramas UML del sistema:
- Diagrama de caso de uso
- Diagrama de actividad
- Diagrama de clases
- Diagrama de componentes

### frontend/
Contiene las vistas de diseño de la interfaz de usuario (mockups).

### dod/
Incluye el documento Definition of Done (DOD), que establece los criterios de validación del diseño.

---

## Metodología

Se utilizó la metodología ágil Scrum para la organización del proyecto, permitiendo una planificación iterativa mediante:
- Product Backlog
- Sprint Backlog
- Roadmap
- Uso de tablero en Trello

---

## Herramientas utilizadas

- Figma (diseño de vistas)
- Draw.io (diagramas UML)
- GitHub (control de versiones)
- Trello (gestión de tareas)

---

## Objetivo

Diseñar una solución estructurada y funcional para la gestión de reservas hoteleras, aplicando buenas prácticas de análisis y diseño de software.
