# Sistema de Reservas Hoteleras Pacific Reef

Este proyecto corresponde al diseño de un sistema de gestión de reservas hoteleras, desarrollado en el contexto de la asignatura de Ingeniería de Software.

## Descripción del sistema
Sistema de reservas hoteleras que permite a usuarios registrarse, buscar habitaciones, reservar y realizar pagos.

## Arquitectura
El sistema se diseñó bajo una arquitectura de componentes separando frontend, backend y base de datos.

## UML
Se modelaron las vistas:
- Casos de uso
- Actividad
- Clases
- Componentes

## Testing
Se diseñaron casos de prueba para validar:
- Registro
- Login
- Búsqueda
- Reserva
- Pago

## DOD
Se definieron criterios de calidad:
- Seguridad
- UX
- Rendimiento

- ## Coherencia del sistema

Las historias de usuario definidas en Jira se modelaron mediante diagramas UML, los cuales fueron implementados a nivel lógico en clases y posteriormente validados mediante casos de prueba.

Esto asegura trazabilidad completa entre requerimientos, diseño y validación.
